# Agent Development Kit (ADK) Samples

## Overview
This repository contains sample agents built with Google's Agent Development Kit (ADK). It provides ready-to-use agent implementations across multiple programming languages to accelerate development.

## Repository Purpose
- **Official Google ADK samples** for building AI agents
- **Multi-language support**: Python, TypeScript, Go, and Java
- **Production-ready examples** covering various use cases
- **Learning resource** for ADK development patterns

## Key Components

### Python Agents (`python/agents/`)
Extensive collection of specialized agents:
- **Business**: customer-service, financial-advisor, marketing-agency, personalized-shopping
- **Research**: academic-research, fomc-research, deep-search
- **Data**: data-engineering, data-science, plumber-data-engineering-assistant
- **Content**: blog-writer, brand-search-optimization, short-movie-agents
- **Technical**: software-bug-assistant, machine-learning-engineering, RAG, llm-auditor
- **Healthcare**: medical-pre-authorization
- **Payment**: antom-payment
- **Real-time**: realtime-conversational-agent, gemini-fullstack
- **Analysis**: google-trends-agent, image-scoring
- **Multi-agent**: camel, travel-concierge
- **Security**: safety-plugins

### TypeScript Agents (`typescript/agents/`)
- customer_service - Customer support agent implementation

### Go Agents (`go/agents/`)
- llm-auditor - LLM auditing and monitoring agent

### Java Agents (`java/agents/`)
- software-bug-assistant - Bug analysis and resolution helper
- time-series-forecasting - Time series prediction agent

## Technical Stack
